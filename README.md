# グランドワン合同会社 公式ウェブサイト

川崎市幸区を拠点とする軽貨物配送会社「グランドワン合同会社」の公式ウェブサイトです。

## 🚀 技術スタック

- **フレームワーク**: Next.js 16 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **アニメーション**: Framer Motion
- **フォーム管理**: React Hook Form + Zod
- **メール送信**: Resend
- **UI**: Lucide Icons, Sonner (Toast)

## 📦 セットアップ

```bash
# 依存関係のインストール
npm install

# 環境変数の設定
cp .env.example .env.local
# .env.localにResend APIキーを設定

# 開発サーバーの起動
npm run dev
```

## 🌐 デプロイ

このプロジェクトはVercelにデプロイされています。

環境変数:
- `RESEND_API_KEY`: Resend APIキー
